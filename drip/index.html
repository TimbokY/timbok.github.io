常用命令行记录

npm

1.查看所有高级的npm moudles

npm list –depth=0

2.查看所有全局安装的模块

npm list –depth=0 -global

Mac

1.创建文件夹

mkdir ‘name’

2.创建文件

touch ‘name.txt’

3.查看文件目录下的文件

ls 查看普通文件

ls -a 可查看前缀为.的文件

4.打开文件夹

用app打开

open test.txt

用vim打开

  • 键入i: 进入编辑模式
  • 键入esc: 退出编辑模式，进入命令模式
  • 在命令模式下 q 退出程序 w 保存文件

5.安装on-my-zsh

sh -c “$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

6.查看ip

ifconfig

查看硬件和其对应的缩写。

networksetup -listallhardwareports

7.查看当前路径

pwd

8.删除文件

rm test.txt

9.查看所有进程

ps

10.终止进程

kill pid

git

  • –global 使用全局配置文件
  • –system 使用系统级配置文件
  • –local 使用仓库级配置文件
  • –worktree 使用工作区级别的配置文件

1.查看git配置

git config –list

2.配置姓名和邮箱

git config user.name Timbok

git config user.email myemail@qq.com

3.记住远程git仓库账号密码

永久记录

git config –global credential.helper store

自定义记住时间 - 1小时

git config credential.helper ‘cache –timeout=3600’
